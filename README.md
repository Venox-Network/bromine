# Bromine Essentials

Plugin made for random Minecraft-related Venox stuff

**🐛 Bugs / 💡 Suggestions:** Please [open an issue](https://github.com/Venox-Network/bromine/issues/new/choose) to report a bug or suggest an idea

**🆘 Support:** Please [join the Discord](https://venox.network/discord) to get support

## Features

- Disables `/ban-ip` (so ops can't get player IPs) with allowed players list in config
  - Disables it in chat, command blocks, and console (safety side effect for command blocks)

### Removed

These features were removed for the specified reasons:

- Adds `/chattitle [player]` which displays your chat as a title (toggled)
  - **Removal reason:** Moved to [Simple Chat Formatter](https://github.com/srnyx/simple-chat-formatter)
- Adds `/reset [world]` which will regenerate a world (if none is defined it will regenerate `world`, `world_nether`, and `world_the_end`) *[Multiverse-Core](https://spigotmc.org/resources/390) required*
  - **Removal reason:** Broke after recent updates. Too complex to fix for such a niche feature.
